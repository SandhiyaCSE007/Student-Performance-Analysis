# Student Performance Analysis

This project focuses on analyzing student performance using Python and visualizing insights using Power BI.

The project includes:
- A Jupyter Notebook for data cleaning, exploration and analysis
- A Power BI dashboard for interactive visualization

---

## Project Overview

The objective of this project is to understand how different factors such as:
- study hours
- sleep hours
- previous scores
- attendance percentage

affect students' exam performance.

---

## Tools and Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Power BI

---

## Project Files

- Jupyter Notebook file:
  Contains data preprocessing, cleaning, and exploratory data analysis using Python.

- Power BI file:
  Contains an interactive dashboard showing relationships between exam scores and other factors.

---

## Key Analysis Performed

- Data cleaning and preparation
- Exploratory data analysis
- Relationship between study hours and exam score
- Relationship between sleep hours and exam score
- Analysis based on attendance percentage
- Comparison using previous scores

---

## Dashboard Highlights

The Power BI dashboard includes:
- Average exam score
- Attendance percentage overview
- Exam score distribution
- Comparison by sleep hours and previous scores

---

## How to Use This Project

1. Open the Jupyter Notebook to view data cleaning and analysis steps.
2. Open the Power BI file to explore the interactive dashboard.
